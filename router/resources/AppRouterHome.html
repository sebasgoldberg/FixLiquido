Solução para Correção do Preço Liquido de Itens de Pedido

  • Start: Iniciar Daemon.
  • Stop: Parar Daemon.
  • Get Config: Obtenção da configuração de execução atual.
  • Set Config: Modificação da configuração atual (pasar novos valores como query parameters).
  • Reload Config: Faz um reload da configuração. Util em caso que tenha atualizado algum destination.
  • Register Trace: Registrar dados de trace. Pasar as informações do registro de trace no query parameter data.
  • Analysis: Obtem uma lista com o análise por item para o filtro especificado no query parameter filter (exemplo: filter=PurchaseOrder eq '4500000061'). Os dados por item são os seguintes:
    • data: Dados atuais do item.
    • lastFix: Ultimo registro historico de correção.
    • trace.payloadRequest: Payload request associada ao GUID do registro histórico.
    • trace.responseBody: Resposta do calculo de impostos utilizando o trace.payloadRequest com o tipo modificado para gross.
